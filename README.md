# 🌍 Wonder World Gallery

A modern and responsive image gallery built using **React + Vite**.

## ✨ Features

- 📸 Beautiful image gallery layout
- 🎯 Category filtering (Nature, Cities, Wildlife)
- ⚡ Fast performance with Vite
- 🎨 Modern UI with smooth hover effects
- 📱 Fully responsive design

## 🚀 Tech Stack

- React.js
- Vite
- CSS (Custom styling)

## 📂 Project Setup

```bash
npm install
npm run dev
